# Tech-Product-Page
A responsive marketing landing page built using HTML, CSS, and JavaScript for a fictional SaaS product called “Pulse”, a social media monitoring dashboard for brands and creators. The site features multiple sections and a theme toggle that remembers user preferences using localStorage.

## Key Features
Responsive layout optimized for desktop and mobile devices

Clean, modern, and easy-to-navigate interface

Sections for Hero, Features, Testimonials, Pricing, and Call-to-Action

Dark Mode toggle to switch between light and dark themes

Last used theme stored in localStorage for persistent user experience
